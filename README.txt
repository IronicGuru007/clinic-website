How to use:
1. Upload all files and folders exactly as they are.
2. Keep index.html in the repository root.
3. GitHub Pages source should be main + /(root).
4. Replace demo text, phone, email, address, and images with your real clinic details.

Notes:
- This is a static HTML/CSS/JS package.
- Appointment and contact forms are front-end only.
- Main lifestyle photos use remote image URLs for a more premium visual result.
