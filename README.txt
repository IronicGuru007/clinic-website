How to use on GitHub Pages:
1. Create a new GitHub repository.
2. Upload all files and folders exactly as they are.
3. Open repository Settings > Pages.
4. Set branch to main and folder to /root.
5. Save and wait for GitHub Pages to publish.

Important:
- This is a static HTML website package.
- Contact form is frontend demo only.
- Replace clinic name, phone, address, WhatsApp number, email, doctor names, and images.
